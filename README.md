# Данные о бюджетах субъектов РФ

Структурированные и подробные данные о доходах и расходах бюджетов регионов с 2025 года с портала [iminfin.ru](https://www.iminfin.ru). Данные обновляются ежемесячно.

## 📥 Скачать данные

Последний релиз доступен в разделе **[Releases](../../releases/latest)**.

Файлы доступны в форматах:
- **Parquet** — рекомендуется для работы в Python/R
- **CSV** — универсальный формат
- **Excel** — для просмотра и анализа данных в привычных таблицах

## Структура данных

### Доходы бюджетов (`budget_income`)

Данные о доходах консолидированных бюджетов субъектов РФ.

| Колонка | Описание |
|---------|----------|
| `year` | Год |
| `month` | Месяц |
| `income_level` | Уровень категории доходов (от 0 до 5), где 0 — всего доходов, 5 — самая детальная разбивка. |
| `income_part` | Категория доходов. Всего в наборе данных представлено более 3,8 тысяч категорий. |
| `plan` | План по доходам |
| `adj_plan_consolidated` | Уточненный план по доходам консолидированного регионального бюджета |
| `adj_plan_regional` | Уточненный план по доходам регионального бюджета |
| `adj_plan_growth_rate` | Темп уточненного план к исполнению прошлого года |
| `execution_consolidated` | Исполнено на первое число следующего месяца по консолидированному бюджету |
| `execution_regional` | Исполнено на первое число следующего месяца по региональному бюджету |
| `growth_rate_regional` | Темп роста исполнения к соответствующему периоду прошлого года по региону |
| `growth_rate_federal_district` | Темп роста исполнения к соответствующему периоду прошлого года по федеральному округу |
| `growth_rate_russia` | Темп роста исполнения к соответствующему периоду прошлого года по стране |
| `object_name` | Название географической единицы (страна, федеральный округ, регион) |
| `okato` | Код ОКАТО |
| `oktmo` | Код ОКТМО |
| `object_level` | Уровень географической (страна, федеральный округ, регион) |

### Расходы бюджетов (`budget_expense`)

Данные о расходах консолидированных бюджетов субъектов РФ.

| Колонка | Описание |
|---------|----------|
| `year` | Год |
| `month` | Месяц |
| `expense_level` | Уровень детализации категорий расходов:
0 — Расходов всего
1 — Расходы по разделам бюджетной классификации (например, «Здравоохранение»)
2 — Расходы по подразделам бюджетной классификации (например, «Скорая медицинская помощь») |
| `expense_part` | Название раздела или подраздела бюджетной классификации расходов |
| `plan` | План по расходам |
| `adj_plan_consolidated` | Уточненный план по расходам консолидированного регионального бюджета |
| `adj_plan_regional` | Уточненный план по расходам регионального бюджета |
| `adj_plan_growth_rate` | Темп уточненного плана к исполнению прошлого года |
| `execution_consolidated` | Исполнено на первое число следующего месяца по консолидированному бюджету |
| `execution_regional` | Исполнено на первое число следующего месяца по региональному бюджету |
| `growth_rate_regional` | Темп роста исполнения к соответствующему периоду прошлого года по региону |
| `growth_rate_federal_district` | Темп роста исполнения к соответствующему периоду прошлого года по федеральному округу |
| `growth_rate_russia` | Темп роста исполнения к соответствующему периоду прошлого года по стране |
| `object_name` | Название географической единицы (страна, федеральный округ, регион) |
| `okato` | Код ОКАТО |
| `oktmo` | Код ОКТМО |
| `object_level` | Уровень географической единицы (страна, федеральный округ, регион) |

## Обновления

Данные обновляются автоматически **5-го числа каждого месяца**. Новые релизы публикуются с тегом версии `vYYYY.MM`.

Следите за обновлениями:
- ⭐ Добавьте репозиторий в избранное (Star)
- 👁 Подпишитесь на релизы (Watch → Custom → Releases)
